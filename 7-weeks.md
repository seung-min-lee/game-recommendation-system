### 2. LightGCN 그래프 개선

#### 2-1. 선 가시성 개선
- 엣지를 3종 단일 trace로 묶어 렌더링 최적화
  - 타 유저 연결선: **노란색** `rgba(255,220,50,0.35)`
  - 내 보유 게임 연결선: **파란색** `rgba(120,180,255,0.70)`
  - 내 추천 게임 연결선: **빨간색** `rgba(255,80,80,0.90)`
- 높이 700px, 배경색 대비 개선으로 선 뚜렷하게

#### 2-2. 추천 게임 별도 컬럼 배치
- 기존 단일 세로 배치 → **3열 레이아웃**으로 변경
  - `x=0` : 유저 노드
  - `x=1` : 보유·기타 게임 노드
  - `x=2` : 추천 게임 노드 (우측 별도 분리, 균등 세로 배치)
- 추천 게임 라벨을 빨간 별(⭐)로 강조

#### 2-3. 추천 게임 이름 `Game_숫자` 표시 버그 수정
- `rec_games` 리스트의 `name` 필드를 우선 사용
- 없을 때만 `GAME_CATALOG` → `Game_{app_id}` 순으로 폴백

### 4. 친구 1:1 비교 패널 추가

#### 4-1. 사이드바 친구 카드 → 버튼
- 각 친구 카드에 **→** 버튼 추가 (이름 오른쪽 인라인 배치)
- 클릭 시 선택 상태(✕)로 토글, 선택된 카드 배경 강조

#### 4-2. 친구 비교 패널 렌더링
- 대시보드 메인 영역에 **나 vs 친구명 비교** 패널 표시
- 구성:
  - 장르 유사도 % 헤더 (유사도 색상 3단계 표시)
  - 요약 카드 4개: 내 플레이타임 / 친구 플레이타임 / 내 보유 / 공통 게임 수
  - 장르 레이더 차트 (나=빨강, 친구=파랑)
  - 공통 보유 게임 플레이타임 비교 바 차트 (상위 10개)
  - 공통 보유 게임 썸네일 그리드 (최대 12개, 클릭 시 Steam 스토어 이동)

### 5. 유사도 계산 방식 변경 (Jaccard → 코사인 유사도)

| 항목 | 이전 | 이후 |

| 문제점 | 장르 1개라도 같으면 동일 기여 | - |
